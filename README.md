# cemu-docs

Documentation for [CEMU](https://github.com/cyyself/cemu)

## How to write?

Environment Setup:

```console
$ python -m venv venv
$ . venv/bin/activate
$ pip install -U pip wheel setuptools
$ pip install -r requirements.txt
```

Preview the docs:

```console
$ mkdocs serve  # make sure you have executed `. venv/bin/activate` first
```
